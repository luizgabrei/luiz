Nessa aula, demos início ao desenvolvimento do nosso projeto, criando o arquivo index.html. Vamos relembrar o que aprendemos?

Criando a pasta do projeto

Antes de tudo, vamos precisar armazenar o projeto numa pasta, dentro do nosso computador.

Então, abra o explorador de arquivos do seu computador e escolha a pasta principal onde vamos criar a pasta do projeto.

Depois clicamos com o botão direito e escolhemos a opção Novo > Pasta.

No caso do instrutor, a pasta se chamará "aluraflix". Após pressionar a tecla Enter, teremos a pasta do projeto criada.

Abrindo a pasta do projeto no VS Code

O próximo passo é começar a escrever arquivos com código que vão descrever a página para que as pessoas possam acessá-la. Para fazer isso, utilizaremos o Visual Studio Code (ou VS Code).

A partir da tela inicial do programa, podemos utilizar o menu lateral esquerdo, seguindo o caminho File > Open Folder….

Por padrão, o VS Code vem no idioma "inglês".

Agora, encontre a pasta do seu projeto e clique em Selecionar Pasta.

Criando o arquivo index.html

Agora estamos prontos para criar a página.

No menu lateral esquerdo, clicaremos no ícone de "novo arquivo", como mostra a figura a seguir, criando um arquivo de nome index.html:

A extensão .html é o padrão de uma página na internet.

Após criar o arquivo, vamos digitar o que queremos na página. Inicialmente, queremos um título, que nesse caso será "ALURAFLIX". Feito isso, vamos salvar em File > Save (ou utilizar o atalho Ctrl + S) para salvar.

index.html

ALURAFLIX

Abrindo no navegador

Em seguida, vamos abrir esse arquivo no navegador.

Para isso, clicamos com o botão direito sobre o arquivo index.html. Dessa forma, encontraremos várias opções de como abrir esse arquivo.

Nesse caso, vamos escolher a opção "Reveal in File Explorer" para revelar no explorador.

Ao fazer isso, o arquivo index.html será exibido na pasta do projeto "aluraflix".

Com um duplo clique nesse arquivo, ele será aberto no navegador. Nesse momento, teremos uma página apenas com a palavra "ALURAFLIX" no canto superior esquerdo.

Já temos o começo da página, apenas com uma palavra. Porém, queremos muito mais, então vamos dar continuidade.

Iniciando o código HTML

De volta ao Visual Studio Code, vamos trabalhar com o código para criar uma página mais complexa e rica. Primeiramente, vamos pegar as informações do vídeo que queremos colocar na página.

O primeiro vídeo que iremos colocar será de um filme sobre o Aranhaverso. Com o YouTube aberto no navegador, vamos acessar o vídeo "ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?".

Queremos usar esse título, então vamos selecionar o texto e copiar (atalho Ctrl + C).

Feito isso, vamos voltar para o Visual Studio Code, teclar Enter algumas vezes para adicionar linhas em branco na página, e colar com o atalho Ctrl + V (ou clicando com o botão direito e selecionando Paste).

Em seguida, na linha abaixo, vamos adicionar uma tag, por exemplo, "#homem-aranha".

ALURAFLIX

ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?
#homem-aranha

Após salvar, podemos voltar ao navegador e atualizar a página (atalho Ctrl + R). Como resultado, teremos o texto "ALURAFLIX ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME? #homem-aranha" em uma única linha.

O navegador entende o documento como um texto corrido, e não é isso que queremos.

Cabeçalho da página

O texto "ALURAFLIX" no topo do código corresponde ao cabeçalho da página. Se é o cabeçalho, vamos escrever a palavra "cabeçalho", mas através de um código, assim como na matemática.

Então, vamos adicionar a tag <header> no nosso documento HTML, inserindo o texto ALURAFLIX dentro dela, desta forma:

<header>ALURAFLIX</header>

ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?
#homem-aranha

header significa "cabeçalho"!

Observe agora o resultado no navegador:

Título da página

Além disso, queremos que a informação da linha 4, ou seja, o texto "ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?", seja um título importante. Para adicionar títulos, usamos outra tag.

Então, adicione a tag <h1> no seu arquivo, inserindo dentro dela o texto "ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?".

<header>ALURAFLIX</header>

<h1>ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?</h1>
#homem-aranha

h1 indica um título de importância nº1, ou seja, o maior e mais importante da página!

Vamos conferir o resultado até o momento? Ao atualizar a página no navegador, teremos um cabeçalho na parte superior contendo o texto "ALURAFLIX", seguido de um título grande na linha abaixo ("ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?"), e de um texto solto na última linha ("#homem-aranha").

Parágrafo da página

O texto "#homem-aranha" ainda está solto no código, pois não utilizamos nenhuma tag. Nesse caso, queremos criar um parágrafo de texto, e também existe uma tag específica para isso.

Então, adicione a tag <p> no seu projeto, inserindo dentro dela o texto "#homem-aranha":

<header>ALURAFLIX</header>

<h1>ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?</h1>
<p>#homem-aranha</p>

p indica um "parágrafo"!

De volta ao navegador, após atualizar novamente a página, teremos o cabeçalho (<header>), o título principal (<h1>), e o parágrafo (<p>), todos com os textos desejados.

Adicionando o vídeo

Nesse momento, temos a primeira versão da página, mas ainda falta algo muito importante: o vídeo.

Para começar, vamos acessar a página do vídeo no YouTube. Logo abaixo do vídeo, encontramos o botão de compartilhar (Share).

Como queremos colocar o vídeo dentro da nossa página, o objetivo é embutir. A opção para isso se chama Embed, representada pelo símbolo da tag que acabamos de conhecer (<>).

Após clicar nessa opção, teremos acesso a um código grande à direita. Vamos selecioná-lo e clicar no botão Copy no canto inferior direito para copiar.

Uma vez copiado o código, precisamos colar ao código no Visual Studio Code, vamos colar após a tag <p> da linha 5.

Lembre-se: no HTML, não importa a quantidade de espaços entre uma linha e outra.

<header>ALURAFLIX</header>

<h1>ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?</h1>
<p>#homem-aranha</p>

<iframe width="560" height="315" src="https://www.youtube.com/embed/gt_fAE1Eg2Q?si=EEv-tsY_b1B2OwKE" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>

Com isso, temos o arquivo completo. Podemos salvar com o atalho Ctrl + S, voltar para o navegador, e atualizar a página com Ctrl + R.

Como resultado, temos o cabeçalho, o título, o parágrafo e o vídeo.

Se apertarmos o botão de play no vídeo, ele será reproduzido na nossa própria página.

Definindo o corpo da página

Como todo o conteúdo de código que construímos é a página HTML, vamos colocar isso dentro do corpo da página.

Para as tags <header>, <h1>, <p> e <iframe> não ficarem soltas na página, vamos utilizar uma tag chamada <body> ("corpo" em inglês) na primeira linha do código. Para fechar, adicionamos a tag </body> ao final do arquivo, na última linha.

<body>
    <header>ALURAFLIX</header>

    <h1>ATRAVÉS DO ARANHAVERSO SUPERA O PRIMEIRO FILME?</h1>
    <p>#homem-aranha</p>

    <iframe width="560" height="315" src="https://www.youtube.com/embed/gt_fAE1Eg2Q?si=EEv-tsY_b1B2OwKE" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>

</body>

Após salvar, não haverá nenhuma diferença no navegador, mas isso é importante para que a página fique bem escrita. Todo o conteúdo está dentro do corpo, e é dessa forma que definimos a página HTML.
